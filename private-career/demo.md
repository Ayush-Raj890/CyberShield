# Demo Guide

Use this as the short walkthrough for presentations and evaluations.

## Guest Flow

1. Open the home page.
2. Use the AI detector on a sample suspicious message.
3. Browse public reports and articles.

## User Flow

1. Register a new account.
2. Verify the OTP.
3. Log in.
4. Submit a report.
5. Open the dashboard and edit profile settings.

## Admin Flow

1. Log in with an admin account.
2. Review pending moderation items.
3. Suspend or unsuspend a user.
4. Review reports and client error logs.

## Demo Notes

- Keep screenshots ready for home, AI result, dashboard, and admin moderation.
- Use seeded demo data so the app never looks empty.
- Prefer short explanations over deep implementation details during live demos.
